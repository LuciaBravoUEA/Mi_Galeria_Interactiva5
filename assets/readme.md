# Galería interactiva con JavaScript 
 Esta tarea consta en una galería de imágenes con animación  desarrollada utilizando HTM, CSS y JavaScript con DOM.
 al seleccionar las imagenes se observa un relieve que sobre sale la imagen, también cada imagen al darle el toque tiene 
 el borde de color.
 los botones de la galeria tiene un color llamativo 
 el titulo lo tenemos centrado 

 # Funcionalidades
 - Galería de 8 ímagenes 
 - Agregar nuevas fotos mediante la URL
 - Seleccionar una imagen haciendo clic
 - Resaltar la imagen seleccionada
 - Eliminar la imagen seleccionada usando un botón o la tecla Delete.

 # Se uso:
 - HTML
 - CSS3
 - JavaScript
